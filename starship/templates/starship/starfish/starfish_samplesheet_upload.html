This is an alternative to adding genomes one by one. For single genomes, use the "Add Genome" button instead.

Samplesheet Format

The samplesheet should be a CSV file with the following columns:

Required Columns:
  • genomeID: Unique identifier for the genome
  • fna: Path to genome assembly file (FASTA)
  • gff3: Path to GFF3 annotation file
Optional Columns:
  • taxID: NCBI taxonomy ID
  • emapper: Path to eggNOG mapper annotations
  • cds: Path to CDS sequences (FASTA)
  • faa: Path to protein sequences (FASTA)
Example:
genomeID,taxID,fna,gff3,emapper,cds,faa
genome1,12345,/path/to/genome1.fasta,/path/to/genome1.gff3,,,
genome2,67890,/path/to/genome2.fasta,/path/to/genome2.gff3,/path/to/emapper.annotations,,
