About Starfish

The starfish-nextflow pipeline automates the detection and analysis of Starships, which are large mobile genetic elements found in fungal genomes.

Key Features:
  • Automatic file formatting
  • Parallel processing
  • Functional annotation
  • Element detection
  • Visualization
Required Input:
  • Genome assembly (FASTA)
  • GFF3 annotation file
  • Optional: eggNOG annotations
Pipeline Parameters
Gene Models:
  • tyr: Tyrosine recombinase (default)
  • nlr: NLR proteins
  • fre: FRE proteins
  • plp: PLP proteins
  • duf3723: DUF3723 domain
Filtering Parameters:
  • Missing: Max missing genes in orthogroup
  • Max Copy: Max copy number in orthogroup
  • PID: Min percent identity for BLAST
  • HSP: Min HSP length for BLAST
