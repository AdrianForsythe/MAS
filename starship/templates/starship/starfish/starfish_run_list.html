About Starfish

Starfish is a computational pipeline for detecting and analyzing Starships in fungal genomes.

What it does:
  • Detects starship elements
  • Analyzes genomic regions
  • Generates visualizations
  • Exports results
Input requirements:
  • Genome assembly (FASTA)
  • Genome annotation (GFF3)
  • Optional: Pre-computed annotations
